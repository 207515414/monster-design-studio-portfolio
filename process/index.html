Home / Process

From project brief to agreed file delivery.

1. Share drawings and project brief

Send the available plans, elevations, models, reference images, required service and target date.

2. Confirm scope, schedule and quotation

We review available material and identify deliverables, milestones, file formats and a consolidated review approach.

3. Start a paid pilot or confirmed package

A paid pilot can establish working expectations before a broader scope is confirmed.

4. Produce and review

Production follows the agreed brief. Consolidated feedback keeps review decisions clear.

5. Deliver approved files

Approved files are supplied in the agreed format. Monster CG provides remote production support, not local approval, engineering certification or professional stamping.
