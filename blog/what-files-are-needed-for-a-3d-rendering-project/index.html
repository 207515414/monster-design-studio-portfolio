What files are needed for a 3D rendering project?

The most useful starting package combines current drawings, dimensions, model files if available, material direction, visual references and a list of required views. Not every project starts with a complete set, but identifying gaps early makes scope clearer.

Start with the available design record

Plans, elevations, sections and current design notes help establish the spatial basis. Reference images clarify intended atmosphere without replacing project-specific information.
